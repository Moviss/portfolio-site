O tej stronie

Ta strona powstała zgodnie z metodologią BEM. Reguły CSS zostały pogrupowane w odrębne moduły. Jako preprocessora użyłem Post CSS.

Do automatyzacji procesów użyłem Gulpa, a do optymalizacji i podziału na moduły plików JavaScript Webpacka oraz Babel. Wszystkie pakiety były pobierane i instalowane za pomocą npm.

Obrazy zostały zoptymalizowane pod kątem Reponsive Web Design - występują w kilku wersjach zarówno pod względem rozdzielczości jak i kadrowania. Użyłem również lazysizes aby zapobiec ładowaniu niepotrzebnych obrazów.

W procesie produkcji wersji dystrybucyjnej użyłem również optymalizacji obrazów, html, styli oraz skryptów (usemin, imagemin, cssnano, uglify).
